Installationsanleitung:
_____________________________

Information: Bitte lege vor dem installieren eine Sicherungskopie deines Spieles an!

Schritt 1:
Kopiere alle Daten in das root-Verzeichnis deines Games:

login.css                         --> root/login.css
templates/OpenGame/login_body.tpl --> root/templates/OpenGame/login_body.tpl
images/                           --> root/images/

Schritt 2:
Gehe nun auf die Loginseite deines Games und kontrolliere, ob der Login erfolgreich installiert wurde!
